# Google Play Store - Advanced Analytics Dashboard
A comprehensive EDA project focused on market trends and app performance metrics.

## 🚀 Live Dashboard
[View Live Report on NBViewer](https://nbviewer.org/github/A-sonwane/EDA-project/blob/main/googleplaystore_analysis.ipynb)

## 📊 Internship Task Implementations
- **Interactive Time-Gate:** Dashboard access restricted to 6:00 PM - 9:00 PM IST for secure reporting.
- **Localization:** Automated translation of categories into Hindi, Tamil, and German.
- **Growth Analytics:** Visual highlighting of apps with >25% Month-over-Month install growth.
- **Advanced Filtering:** Multi-layer filters excluding apps with numeric names and specific ratings.

## 🛠️ Tech Stack
- **Language:** Python 3.10
- **Libraries:** Pandas (Data Wrangling), Matplotlib/Seaborn (Visualization), Pytz (Timezone handling).
